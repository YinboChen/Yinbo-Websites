Slide Controller

members: Yinbo Chen, Mutian Yan

This project is an experimental project which is for testing the communication between various devices on multiple operating systems. To achieve this purpose, my partner and I used Java as a fundamental coding language that can be performed on multi-platform, such as macOS and Win. In this project, we used the smart devices as a controller to remote the slides on a terminal via a standardized Bluetooth channel.
